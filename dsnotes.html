WHAT IS DATA TYPE?

Data Type is noting but setting the DOMAIN of the value. Data type not sets the Domain, it also Define the ALLOWED OPERATIONS on those values.
EXAMPLE :-

  1. INT TYPE:
    • Takes only only Integer Values.
    • OPERATIONS:- Addition, Substraction, Multiplication, Bitwise operations etc.
  2. FLOAT TYPE:
    • Takes only Floating point Values.
    • OPERATIONS:- Addition, Substraction, Multiplication, Division etc(Bitwise and % operations are mot allowed).

USER DEFINED DATA TYPES

The Operations and Values of USER DEFINED datatype are not specified in the language itself but is specified by USER.
EXAMPLE: by using Structure, we are defining our own type by combining other data types.

struct point{
int x;
int y;
}

ABSTRSCT DATA TYPES(ADT)

ADTs are like user define data types which defines opeartions on values using function without specifying what is there inside the functuion and how operations are performed.

ADT provides ABTRACTION. Which means hiding the backend details from the user.
EXAMPLE:-

  • STACK ADT:
    • A Stack consists of element of the same type arrangement in a sequential order.

ADVANTAGES OF ADT:

  • Let say, if someone wants to use the stack in the program, then he can simply use PUSH and POP operations without konwing its implementation.
  • Also, if in future, the implementation of stack is changed form array or linked list, then client program will work the same way without being affected.
